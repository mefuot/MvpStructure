# MvpStructure
MVP layout base class for Android

## Install
### Maven
```groovy
<dependency>
  <groupId>com.pong.library</groupId>
  <artifactId>mvp-structure</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
### Gradle
```groovy
compile 'com.pong.library:mvp-structure:1.0.0'
```
